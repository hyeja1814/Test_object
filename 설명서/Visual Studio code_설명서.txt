편집기 : 파일을 열어 코드를 추가 및 수정할 수 있는 탭. 
텍스트 에디터의 가장 핵심 기능임.

탐색기 : 코딩 작업을 진행할 폴더 트리 및 열려 있는 파일을 볼 수 있슴.
필요한 파일을 하나씩 열지 말고 프로젝트 폴더를 한 번에 열어 놓고 작업하면 편함

검색 : 열려 있는 폴더 대상으로 검색할 수 있습니다. 특정 폴더를 빼고 검색한다거나,
특정 단어를 한 번에 수정하는 등 편리하게 사용할 수 있음 / 단축기 ctrl + shift + f

Git : 기본적으로 Git GUI 플러그인을 제공하고 있어서 소스트리를 사용하지 않고도
간단한 버전 관리 작업 완료 가능

디버그 : 코드를 돌려보고 각 변수에 어떤 값이 담겨 있는지, 에러는 어디서 나는지 확
인할 수 잇슴. 개발하는 언어에 따라 환경 설정이 필요

확장 : 코드 문법 검사, 자동정렬 등 여러 가지 유용한 확장 플러그인을 설치할 수 있음.
직접 확장 프로그램을 만들어 출시할 수도 있음.

통합 터미널 : 컴퓨터에 텍스트로 명령을 내릴 수 있는 터미널을 비주얼 스튜디오 코드 
내부에서 사용할 수 있는 기본 플러그. 상단 메뉴바에서 [보기 - 통합 터미널]을 선택하면
볼 수있음. 이거 사용시 앱 사이를 전환하는 불편함 줄일수 있음

편집기 분할 : 코드 편집기를 상하좌우로 분할해서 한번에 여러 파일을 볼 수 있음
