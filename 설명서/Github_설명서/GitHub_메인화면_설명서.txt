대시보드 컨텍스트 : 원격저장소를 개인 계정에만 속하도록 만들 수 있지만,
회사나 단체이름으로 단체 계정을 만들어 관리 가능. 전환 버튼을 이용해
메인화면을 어떤 계정 기준으로 표시할지 선택.

저장소 목록(Repositories) : 현재 기여하고 있는 모든 원격저장소 목록이 표시됨, 최근에
 코드를 올린 순서대로 정렬됨. 오른쪽의 [NEW] 버튼을 클릭하여 새 저장소
만들 수 있음.

팀 목록 : 단체 계정 안에 더 작은 분류인 팀()을 만들 수 있음.
내가 속한 팀을 볼 수 있음.

마켓플레이스(Marketplace) : 코드를 GitHub에 올히면 자동으로 빌드해 주거나, 
메신저와 연동할 수 있는 등의 플러그인을 구입할 수 있음

둘러보기(Explore) : GitHub와 관련된 블로그 기사나, 요즘 뜨는 저장소 등을 볼 수 있음

알림(Notifications) : 나와 관련된 모든 이슈, 풀 리퀘스트, 멘션(닉네임 태그) 등이
알림으로 오고, 저장 됨. Watch하는 저장소 소식 볼수 있. 참여하진 않지만 소식받기
유요함.

새로 만들기 : 새 저장소나 단체, 혹인 gist(저장소보다 작은 코드르 올리고 싶을 때
유용. 작은 코드 조각 간편하게 올릴 수 있음)를 만들 수 있다.

최근 활동(Recent activity) : 내가 팔로우하는 저장소의 최근 활등이 보임

모든 활용(All activity) : 내가 팔로우하는 저장소, 팔로유하는 유저의 활동이 보임

뜨는 저장소 보기(Explore repositories) : GitHub에서 큐레이션한 최근 뜨는 원격
저장소들이 보여짐. 요즘 오픈소스 트렌드가 어떤지 확인 가능
